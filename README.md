# SmartStudy Flashcards

A comprehensive study aid application that helps users create, manage, and share interactive flashcards.

## Features

### Core Features
- Interactive flashcard creation (Basic Q/A, Multiple Choice, True/False, Definition, Summary)
- Multiple methods for card creation (Manual, Document Upload, URL Input)
- Study tracking and analytics

### Premium Features
- Themes and personalization
- Unlimited flashcards
- Audio integration
- Image-based flashcards
- Collaboration and sharing
- Premium support

## Getting Started

### Prerequisites
- Python 3.8+
- Node.js 16+
- npm or yarn

### Installation

1. Clone the repository
2. Install backend dependencies:
```bash
cd backend
pip install -r requirements.txt
```

3. Install frontend dependencies:
```bash
cd frontend
npm install
```

### Running the Application

1. Start the backend server:
```bash
cd backend
uvicorn main:app --reload
```

2. Start the frontend development server:
```bash
cd frontend
npm start
```
